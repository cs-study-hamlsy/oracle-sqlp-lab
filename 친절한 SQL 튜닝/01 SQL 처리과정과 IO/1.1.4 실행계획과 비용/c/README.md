# C 디렉터리 안내

이 주제는 SQL 실행과 실행계획 비교가 중심이므로, 별도 C 실습 파일을 직접 유지하지 않습니다.

실행 소스는 [../pro-c/plan_test.pc](/C:/oracle-sqlp-lab/친절한%20SQL%20튜닝/01%20SQL%20처리과정과%20IO/1.1.4%20실행계획과%20비용/pro-c/plan_test.pc) 하나로 관리합니다.

실제 빌드 흐름은 다음과 같습니다.

1. `proc`가 `plan_test.pc`를 C 파일로 전처리합니다.
2. 생성된 C 파일을 C 컴파일러로 컴파일합니다.
3. 최종 실행 파일을 실행합니다.

즉 이 디렉터리에서 따로 실행할 C 파일은 없고, PRO-C 전처리 결과가 로컬 빌드 산출물로만 생성됩니다.
