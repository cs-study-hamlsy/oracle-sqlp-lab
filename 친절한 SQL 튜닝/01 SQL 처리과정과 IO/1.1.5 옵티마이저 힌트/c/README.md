# C 디렉터리 안내

이 주제는 "옵티마이저 힌트가 실행계획에 어떤 영향을 주는가"를 확인하는 것이 중심이다.

그래서 별도 C 소스 파일을 직접 유지하지 않고, [../pro-c/hint_plan_test.pc](/C:/oracle-sqlp-lab/친절한%20SQL%20튜닝/01%20SQL%20처리과정과%20IO/1.1.5%20옵티마이저%20힌트/pro-c/hint_plan_test.pc) 하나를 기준으로 관리한다.

실제 흐름은 아래와 같다.

1. `proc`가 `hint_plan_test.pc`를 C 파일로 전처리한다.
2. 생성된 C 파일을 C 컴파일러로 컴파일한다.
3. 실행 파일을 구동해 실행계획을 비교 출력한다.

즉 이 디렉터리는 "직접 작성하는 C 예제"보다 "PRO-C 전처리 결과가 만들어지는 개념"을 설명하는 용도다.
