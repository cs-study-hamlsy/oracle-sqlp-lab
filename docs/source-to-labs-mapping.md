## 새 실습을 추가할 때 규칙

1. 먼저 `labs/` 아래 표준 실습을 만든다.
2. 필요하면 출처 교재 README나 매핑 문서에 링크를 추가한다.
3. 교재 폴더 아래에는 새 C/Pro*C 구조를 복제하지 않는다.
4. 실습 기본 형식은 SQL-only를 우선한다.
